# Nest-Server-Tools

🦁 帮助 nest-server 开发者在 vscode 中快速创建模板/目录/文件

## Features

右键单击当前项目中的目录后，就可以看到对应的菜单选项，输入指定名称后会自动创建模块/文件/目录。

| `navigation` |
| --------------------------------|
| 🏗  Nest-S: New Module              |
| 🏗  Nest-S: New Extend              |

| `modification` |
| ------------------------------- |
| 📄  Nest-S: module                  |
| 📄  Nest-S: sequelize               |

| `commands` |
| --------------------------------|
| 📦  Nest-S: BO                      |
| 📦  Nest-S: DTO                     |
| 📦  Nest-S: Controller              |
| 📦  Nest-S: Filter                  |
| 📦  Nest-S: Guard                   |
| 📦  Nest-S: Interceptor             |
| 📦  Nest-S: Pipe                    |
| 📦  Nest-S: Provider                |
| 📦  Nest-S: Service                 |
